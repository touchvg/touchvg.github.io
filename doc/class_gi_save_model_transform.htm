详细描述

保存和恢复模型坐标系的变换矩阵的辅助类

利用该类在堆栈上定义局部变量，该变量出作用区后自动析构从而恢复模型坐标系。 利用该类可以避免因中途退出或异常时没有执行恢复模型坐标系的语句。

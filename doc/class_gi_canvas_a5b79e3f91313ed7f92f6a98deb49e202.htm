virtual bool GiCanvas::clipPath ( )
pure virtual

Intersect the current clipping path with the path added by beginPath().

Note that this function resets the context's path to the empty path.

返回
true if successful and the result clipping path isn't empty.
参见
saveClip, restoreClip, beginPath

在 GiGdipCanvas, GiShapeAdapter, GiCanvasAdapter, GiRecordCanvas , 以及 GiSvgCanvas 内被实现.

详细描述

保存和恢复图形系统的剪裁框的辅助类

利用该类在堆栈上定义局部变量，该变量出作用区后自动析构从而恢复剪裁框。 利用该类可以避免因中途退出或异常时没有执行恢复剪裁框的语句。

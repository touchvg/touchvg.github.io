◆ dms2Deg()

float mgbase::dms2Deg ( float  angle)
static

度分秒转换到角度(度)

度分秒形式的角度的整数部分为度，十分位和百分位为分， 千分位和万分位及其他小数位为秒。例如32.251242表示32°25'12.42"。

参数
[in]angle度分秒形式的角度
返回
以度为单位的角度
参见

TouchVG framework
通用矢量绘图框架 编程文档

概述

欢迎使用通用矢量绘图框架(https://github.com/rhcad/touchvg)的编程文档，欢迎反馈意见。
如需Fork或提交PR问题或补丁，请访问 vgcore、 vgios、 vgandroid 等相关项目。

通用图形平台库的跨平台部分采用C++实现，包含下列模块：

  • 几何对象类
    数学几何库的几何对象类，包含矢量、坐标、变换矩阵、矩形框等类。
  • 几何计算函数
    数学几何库的线性代数计算函数。
  • 图形接口类
    图形接口库的各种显示类、符号类，用于屏幕显示、打印。
  • 图形实体类
    图形实体库包含跨平台矢量图形类和交互命令类。
  • 图形序列化类
    图形存取接口，其中的JSON虚列化模块（json）采用rapidjson框架实现。
  • 交互命令类
    图形实体库包含跨平台矢量图形类和交互命令类。
  • 内核视图类
    与各种设备平台衔接的跨平台绘图视图类。

通用图形平台库的设备平台适配器包含视图适配器、画布适配器、辅助访问接口。

  • Windows平台适配器
    Windows适配器采用GDI/GDI+实现。
  • WPF平台适配器
    WPF适配器采用CSharp、WPF实现。
  • iOS平台适配器
    iOS适配器采用Quartz 2D、UIKit框架和ObjC语言实现。
  • Android平台适配器
    Android适配器（android/TouchVG）采用android.graphics、android.view包和Java语言实现。
